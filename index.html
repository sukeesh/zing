Installation

Get started with Zing in minutes

Download Zing

Get the latest version of Zing for macOS.

Download for macOS
Opening on macOS
  1. After downloading, open the DMG file
  2. Drag Zing to your Applications folder
  3. When you first open Zing, macOS may show a security warning
  4. Go to System Preferences → Security & Privacy → General
  5. Click "Open Anyway" to allow Zing to run
  6. You may need to approve a second security prompt

Note: macOS requires approval for apps not from the App Store. This is normal and helps keep your Mac secure. Zing is safe to use.

Build from Source

If you prefer to build Zing from source, follow these instructions:

Prerequisites
  • Rust (latest stable version) and Cargo
  • A C/C++ compiler (for some dependencies)
# Clone the repository
git clone https://github.com/sukeesh/zing.git
cd zing

# Build in release mode
cargo build --release

# Run the application
cargo run --release
